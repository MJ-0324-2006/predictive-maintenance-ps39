# Predictive Maintenance of Industrial Machinery

IBM SkillsBuild AICTE 2026 Internship Project — Problem Statement No. 39

## Overview
A machine learning model built using IBM watsonx.ai Studio AutoAI to predict 
industrial machine failure types (No Failure, Tool Wear, Heat Dissipation, Power Failure) 
based on sensor data (air temperature, process temperature, rotational speed, torque, tool wear).

## Contents
- `dataset.csv` – Kaggle dataset used for training
- `PS39_Problem_Statement.pdf` – Problem statement document
- `PS-39.pptx` – Project presentation
- `Notebook.ipynb` – Python notebook exported from IBM watsonx.ai AutoAI

## Technology
IBM Cloud Lite, Watson Studio, AutoAI, watsonx.ai Runtime

## Result
Best model: Batched Tree Ensemble Classifier — Accuracy: 99.5%
